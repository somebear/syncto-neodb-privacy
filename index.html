SyncToNeoDB Privacy Policy

Last updated: [当前日期]

1. 数据收集

我们只收集以下必要数据：

  • NeoDB API Token（由用户提供）
  • 豆瓣页面内容（书籍/电影信息、用户评分和评论）

2. 数据使用

收集的数据仅用于：

  • 与NeoDB进行身份验证
  • 将用户的豆瓣评分和评论同步到NeoDB

3. 数据存储

  • API Token安全存储在Chrome本地存储中
  • 不保存其他任何用户数据
  • 所有数据仅存储在用户的浏览器中

4. 数据共享

我们不会：

  • 向第三方出售或转让用户数据
  • 将数据用于扩展功能以外的目的
  • 收集任何分析或跟踪数据

5. 用户权限

  • 用户可以随时删除存储的API Token
  • 卸载扩展会自动删除所有存储的数据
